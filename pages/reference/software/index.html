MSP430 programming

  • Simply Embedded's excellent series of programming tutorials for the MSP430. This takes you through setting up a toolchain with the gcc compiler and mspdebug, sets up a makefile with a nice project structure, covers the MSP430 clocks, timers, and interrupts, and more. Great stuff, if somewhat hardcore.
